    ProjextAngularJSPlugin

    This service is in charge of adding the required loader to handle AngularJS inject annotations and updating a target Babel configuration so the loader will work.

    This is the method called when the plugin is loaded by projext. It just gets the events service and registers a listener for the reducer event that handles JS rules for browser targets.

    public updateRules(currentRules: Array, target: Target): Array source

    This method gets called when Projext reduces the JS rules for browser targets. It validates the target, adds the plugin loader and modifies, if necessary, the configuration for Babel.
